# Voc-AB

A mobile vocabulary builder app built with React Native and Expo. Helps users learn and retain new words through multiple study modes and a competitive leaderboard.

## Features

- **Learn It** — flashcard-style vocabulary review
- **Challenge** — test your knowledge with timed questions
- **Time Trial** — race against the clock
- **Dictionary** — look up and save words
- **Leaderboard** — compete with other users
- **Spaced repetition** — SQLite tracks your progress locally

## Stack

React Native · Expo · Firebase (auth + cloud sync) · SQLite (local storage)

## Screens

`Home` · `LearnIt` · `Challenge` · `TimeTrial` · `Dictionary` · `Leaderboard` · `Login` · `Register` · `Settings` · `Tutorial`

## Running

```bash
npm install
expo start
```

Scan the QR code with Expo Go on iOS or Android.
